EVENT-LISTENER

Event-Listener is a collaborative web app project that enables users to search for various events in and around their area. It lets users to save events, send events to other users, and receive email notifications for saved events happening in the next 24 hours. Technologies implemented include JavaScript, PostgreSQL and MongoDB to store users, SendGrid for notification, Node.js and Express for back-end, and GoogleOAuth for authenticating.

CROSSWORD GENERATOR

This project focuses on the implementation of computer science concepts and algorithms. With the utilization of the AC-3 algorithm(Arc Consistency),it allows to determine whether the grid is consistent or there are word patterns which cannot be instantiated with words. It prevents having to search the entire grid for a word pattern to backtrack to when some word pattern cannot be instantiated. It searches each word from our built-in dictionary for words that can fit to the “across” and “down” word patterns and for intersecting word patterns.
